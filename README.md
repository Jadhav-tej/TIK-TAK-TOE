Tic Tac Toe Game
This is a simple Tic Tac Toe game built using HTML, CSS, and JavaScript. The game allows two players to take turns marking spaces on a 3x3 grid with "X" and "O". The goal of the game is to align three of the same symbol either horizontally, vertically, or diagonally to win. If all nine spaces are filled without a winner, the game ends in a tie.

Features
Interactive gameplay: Players alternate between "X" and "O" by clicking on the grid.
Win detection: The game automatically detects when a player has won and displays a congratulatory message.
Tie condition: If all squares are filled and no player has won, the game declares a tie.
Reset functionality: Players can reset the game at any time to start a new match.
Responsive design: The game is designed to be played on both desktop and mobile devices.
Technologies Used
HTML5 for the structure of the game.
CSS3 for styling the layout.
JavaScript for game logic and interactivity.
